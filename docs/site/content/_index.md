---
title: "LegacyPodClaw"
---

Formerly known as ClawPod. A native Objective-C port of [OpenClaw](https://openclaw.ai) for jailbroken iOS 6 devices. ~19,000 lines of hand-written code with one vendored dependency ([wolfSSL](https://www.wolfssl.com/) for TLS 1.2). Everything else is built for the memory constraints of legacy iOS 6 hardware.
